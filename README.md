## OOPScape - Игра

Здравейте! Казвам се Ангел Писарев и това е конзолна **лабиринтна игра**, написана на **C++** като университетски проект за **първи курс** (специалност Софтуерно инженерство). Играта върви в **конзолата** и е създадена във **Visual Studio**.

## Какво можете да правите

- Управление на герой (**Knight**) в лабиринт зареден от текстов файл.
- Избягване на противници, които ви преследват автоматично.
- Използване на специална способност – **щит** (команда `OOP`).
- Внимавайте за **капани**, които нанасят щета.
- Преминаване през **4 нива** с нарастваща трудност.

## Как се играе

- **Цел:** Достигнете изхода **F**, преди противникът да ви настигне.
- **Команди:**

| Команда | Действие |
|---------|----------|
| `U` | Движение нагоре |
| `D` | Движение надолу |
| `L` | Движение наляво |
| `R` | Движение надясно |
| `OOP` | Активиране на щит (3 хода неуязвимост) |
| `Q` | Изход от играта |

- **Символи на картата:**

| Символ | Значение |
|--------|----------|
| `H` | Герой |
| `K` | Герой с активен щит |
| `E` | Противник |
| `F` | Изход |
| `T` | Капан (-25 HP при стъпване) |
| `*` | Стена |

## Разширения

- **Knight герой със щит** – временна неуязвимост чрез командата `OOP`.
- **ChaserEnemy с BFS** – противникът винаги намира най-краткия път до героя.
- **Капани** – клетки, нанасящи щета, блокируеми от щита.
- **Fog of War** – виждате само клетките в радиус 3. Вече посетените клетки остават видими.
- **Последователни нива** – след победа на едно ниво автоматично се зарежда следващото.

| Файл | Описание |

| `Knight.h/cpp` | Герой със щит способност |
| `Enemy.h` | Абстрактен базов клас за противници |

| `Trap.h` | Капан |
| `BoardUtils.h` | Помощна функция за проходимост |
| `ConsoleUtils.h` | Цветове в конзолата |
| `Game.h/cpp` | Главен игрови цикъл |
| `OOPScape.cpp` | Входна точка |
